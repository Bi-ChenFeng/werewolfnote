# 狼人杀对局记录 — 项目文档

> 最后更新：2026-04-07

---

## 项目简介

**狼人杀对局记录（werewolfnote）** 是一个全栈 Web 应用，用于记录狼人杀对局中的发言、角色、身份验证、投票及评价，支持 AI 辅助分析和历史对局管理。

- **GitHub：** https://github.com/Bi-ChenFeng/werewolfnote
- **线上前端（静态）：** https://bi-chenfeng.github.io/werewolfnote/
- **后端：** 本地运行，暂未部署到服务器

---

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | 原生 HTML / CSS / JavaScript（单文件，无框架） |
| 后端 | Node.js + Express 5 |
| 数据库 | MySQL（mysql2 驱动，连接池 10） |
| 认证 | JWT（Bearer Token，有效期 30 天） |
| 密码 | bcryptjs 哈希存储 |
| AI | OpenAI Whisper（语音转文字）+ GPT-4o-mini（发言分析） |

### 后端依赖

```
express       HTTP 框架
mysql2        MySQL 驱动（Promise）
bcryptjs      密码哈希
jsonwebtoken  JWT 生成/验证
cors          跨域处理
dotenv        环境变量加载
```

## 数据库结构

数据库名：`werewolf`

```sql
-- 用户表
users (
  id           INT AUTO_INCREMENT PRIMARY KEY,
  username     VARCHAR,
  password     VARCHAR,   -- bcrypt hash
  created_at   DATETIME
)

-- 对局表
games (
  id           INT AUTO_INCREMENT PRIMARY KEY,
  user_id      INT,
  title        VARCHAR,
  board_type   VARCHAR,
  data         JSON,      -- 完整对局状态
  created_at   DATETIME
)
```

## API 端点

### 用户认证

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/register` | 注册（username, password） |
| POST | `/api/login` | 登录 → 返回 JWT token |
| POST | `/api/change-password` | 修改密码（需认证） |

### 对局记录（均需 Bearer Token）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/games` | 获取当前用户全部对局列表 |
| POST | `/api/games` | 保存新对局（title, board_type, data） |
| GET | `/api/games/:id` | 获取单局完整数据 |
| PUT | `/api/games/:id` | 更新对局（title, board_type, data 全量） |
| PATCH | `/api/games/:id` | 仅改名（title） |
| DELETE | `/api/games/:id` | 删除对局 |

### 管理员（需 x-admin-secret 请求头）

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/admin/login` | 管理员验证 |
| GET | `/api/admin/users` | 查看全部用户及对局数 |
| DELETE | `/api/admin/users/:id` | 删除用户（及其所有对局） |
| GET | `/api/admin/games` | 查看全部对局 |
| DELETE | `/api/admin/games/:id` | 删除任意对局 |

## 前端页面说明

### index.html — 登录/注册

- 登录/注册 Tab 切换
- 登录成功 → 跳转 `app/dashboard.html`
- JWT token 存入 `localStorage`（key: `ww_token`）

### app/dashboard.html — 用户主页

- 统计卡片：总对局数、本月对局数、最近活跃
- 常用板子统计（条形图）
- 历史对局列表：支持 **改名**、保存 PDF、删除
- 个人设置弹窗：修改密码 / OpenAI API Key / 默认对局名格式

### app/game.html — 对局记录主界面

**支持板子（22 种）：**

| 人数 | 板子 |
|------|------|
| 6人 | 基础 |
| 9人 | 基础 |
| 10人 | 奇迹商人、月坠光渊、速推场、狼美人骑士、镜隐迷踪、暗夜星辰 |
| 12人 | 奇迹商人、标准场、月坠光渊、丘比特场、狼美人骑士、镜隐迷踪、暗夜星辰、石像鬼守墓人、赤月猎魔人、咒狐乌鸦、孤独少女、猎日逐光、觉醒狼王、觉醒孤独少女、觉醒女巫、狼王摄梦人、狼王守卫 |

**主要功能模块：**

| 模块 | 说明 |
|------|------|
| 警下快捷面板 | 警长提名快捷按钮 |
| 玩家状态栏 | 所有玩家概览（存活/阵营/评价） |
| 玩家卡片 | 每位玩家：角色、警长状态、警徽流、预言家验证、发言、评价 |
| 投票记录 | 多轮投票，弹窗录入，紧凑摘要展示（支持自投/弃票） |
| 发言汇总 | 各玩家发言聚合，支持一键复制和 AI 全场分析 |
| 结算面板 | 选定结局（好人胜/狼人胜/平局）+ 逐人真实身份 |
| 对局存档 | 登录后可保存到服务器，也可载入历史对局 |

**AI 功能（需配置 OpenAI API Key）：**
- 单人发言 AI 总结（GPT-4o-mini）
- 全场 AI 分析，含推荐投票目标
- 语音录入 → 自动转文字（Whisper）

**5 级评价体系：** 明狼 / X偏下 / X / X偏上 / 明好人

### app/admin.html — 管理员面板

- 输入管理密码进入
- 查看/删除全部用户
- 查看/删除全部对局

## 本地存储（localStorage）

| Key | 说明 |
|-----|------|
| `ww_token` | JWT 登录令牌 |
| `ww_user` | 当前登录用户名 |
| `ww_state` | 当前对局状态（自动保存） |
| `ww_board` | 当前选中板子 |
| `ww_load_game` | 待载入的对局数据 |
| `ww_load_game_id` | 待载入的对局 ID |
| `ww_load_game_title` | 待载入的对局标题 |
| `ww_api_key` | OpenAI API Key |
| `ww_naming_prefs` | 对局命名偏好设置 |
