# 狼人杀对局记录助手

一个纯前端的狼人杀对局记录工具，无需安装，打开即用。

## 功能

- **多种板子**：支持 6/9/10/12 人多种经典板子
- **发言记录**：手动记录每位玩家的发言内容
- **AI 转录**（可选）：接入 OpenAI Whisper，自动将录音转为文字
- **AI 分析**（可选）：接入 GPT，自动提取跳神身份、验人结果、警徽流、玩家评价
- **身份标记**：为每位玩家标记身份、存活状态
- **评价系统**：支持「明狼 / X偏下 / X / X偏上 / 明好人」五档评价
- **发言汇总**：底部汇总所有玩家的发言分析，一览全局
- **本地存储**：所有数据保存在浏览器本地，刷新不丢失

## 使用方式

### 在线使用

访问：[https://bi-chenfeng.github.io/-/](https://bi-chenfeng.github.io/werewolfnote/)

### 本地使用

1. 下载 `werewolf.html`
2. 双击打开即可

### AI 功能（可选）

在页面顶部输入 OpenAI API Key，即可启用录音转录和 AI 自动分析功能。不填写 Key 也可以正常手动记录。

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 项目介绍首页 |
| `werewolf.html` | 主程序（单文件，可独立使用） |

## 技术栈

纯 HTML + CSS + JavaScript，无任何依赖。AI 功能使用 OpenAI Whisper 和 GPT-4o-mini。
